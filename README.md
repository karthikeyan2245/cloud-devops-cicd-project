# Cloud DevOps CI/CD Pipeline on AWS

This project demonstrates an end-to-end Cloud DevOps CI/CD pipeline using modern DevOps tools and AWS services.

## Tech Stack
- AWS
- Git & GitHub
- Docker
- Jenkins
- Kubernetes (EKS)
- Terraform

## Project Overview
- Developed a simple HTML application
- Containerized the application using Docker and Nginx
- Version controlled using Git and GitHub
- CI/CD pipeline and cloud deployment will be implemented
